# 游戏合集 v0.1

一款包含扫雷、数独、俄罗斯方块、迷宫、贪吃蛇、五子棋、国际象棋等多款经典小游戏的单机游戏合集。

## 环境要求

- Python 3.8+
- Pygame 库

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行游戏

```bash
cd D:\game
python main.py
```

## 游戏说明

### 扫雷

- **操作方式**：左键点击揭开格子，右键标记地雷
- **游戏目标**：找出所有地雷即可获胜
- **胜利条件**：揭开所有非地雷格子

### 数独

- **操作方式**：鼠标选择格子，键盘输入数字1-9
- **游戏目标**：使每行、每列、每宫都包含1-9且不重复
- **难度级别**：简单、中等、困难、极难、不可能
- **清空格子**：退格键、Delete键、0键

### 俄罗斯方块

- **操作方式**：
  - 方向键左/右：移动方块
  - 方向键上：旋转方块
  - 方向键下：方块下落
  - 空格键：快速下落
  - P键：暂停/继续
- **游戏目标**：消除满行得分，等级越高下落速度越快

### 迷宫

- **操作方式**：方向键控制移动
- **游戏目标**：用最短路径到达出口

### 贪吃蛇

- **操作方式**：方向键控制移动
- **游戏目标**：吃到更多食物，避开障碍物

### 五子棋

- **操作方式**：鼠标点击落子
- **游戏目标**：黑白交替落子，先连五子获胜
- **AI模式**：默认开启，可与电脑对战

### 国际象棋

- **操作方式**：鼠标点击选择和移动棋子
- **游戏目标**：将死对方王即获胜
- **AI模式**：默认开启，玩家执白方，电脑执黑方

## 通用操作

- ESC键：返回主菜单
- 主菜单点击相应游戏按钮进入游戏

## 多语言支持

支持简体中文、繁体中文、英语。点击右上角语言下拉框切换语言。

    ├── maze.py           # 迷宫游戏

## 按键提示

| 游戏 | 按键 | 功能 |

| 扫雷 | 右键 | 标记/取消标记地雷 |
| 数独 | 鼠标 | 选择格子 |
| 数独 | 数字键1-9 | 输入数字 |
| 数独 | 退格键/0/Delete | 清除数字 |
| 俄罗斯方块 | ← → | 左右移动 |
| 俄罗斯方块 | ↑ | 旋转 |
| 俄罗斯方块 | ↓ | 加速下落 |
| 俄罗斯方块 | 空格 | 快速落地 |
| 俄罗斯方块 | P | 暂停/继续 |
